About Me

Name: Diana Salazar

School: Kennesaw State University

Course: IT 3203 Introduction to Web Development

Academic interest: Information Technology with a strong interest in cybersecurity

I chose this topic because HTTP is one of the most important technologies in web communication. It is also closely related to cybersecurity because many attacks and defensive tools involve inspecting HTTP requests, responses, headers, and session behavior. My research paper explains that understanding HTTP helps students see how web applications work and how security issues appear in real systems.

About This Project

This website is Milestone 1 of the IT 3203 class project. The overall project will continue through later milestones, where the site will be expanded with more content and interactive features. For this stage, the goal is to build a clean informational website using HTML and CSS with multiple pages, shared navigation, and organized content.

  • Created with plain HTML and one shared CSS file
  • Includes six pages with a consistent menu
  • Uses headings, lists, an image, and tables
  • Uses both grid and flex for layout
  • Designed in a simple and clean business style
