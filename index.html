Welcome

The World Wide Web is one of the most important systems in modern life. People use it for school, work, shopping, healthcare, banking, and communication. Behind that everyday experience is a set of technologies that allow browsers and servers to exchange information in a standard way. One of the most important of those technologies is Hypertext Transfer Protocol, better known as HTTP.

This site introduces the relationship between HTTP and the World Wide Web. It explains how the web began, how HTTP messages work, why the protocol is considered stateless, and why secure versions of web communication matter so much today. The topic comes from my Assignment 1 research and is written as a foundation for later project milestones. Key Terms

  • HTTP: The protocol browsers and servers use to communicate.
  • World Wide Web: A system of linked web pages and resources accessed through browsers.
  • Client: Usually a browser that requests data.
  • Server: A computer system that stores and sends web resources.
  • HTTPS: Secure HTTP that uses encryption.
  • Status code: A number that shows the result of a request, such as 200 or 404.
